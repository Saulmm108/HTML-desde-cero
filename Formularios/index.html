ATRIBUTOS DE FORMULARIOS

        -"checked" para seleccioar un checkbox o radio por 
        defaul.

        -"selected" igual que el checked pero en un select.

        -"required" que es cuando un campo sea obligatorio 
        y si no se llena no se puede enviar el formulario

        -"readonly" que es si un campo tenga un valor y no 
        se pueda cambiar

        -"autocomplete" significa que el navegador tiene los 
        datos automaticamente se llenan solitos.
